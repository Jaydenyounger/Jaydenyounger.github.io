## Project Breif
This is my final for my Level and Environmental Design class. We were tasked with creating a real-time environment of a single-room space. I wanted to take inspiration from Dishonored's Clockwork Mansion, where the rooms in the area can dynamically change with an interaction, so I tried to incorporate this into a sci-fi cyberpunk dystopia, where an expensive penthouse suite transforms into a spyroom headquarters.

The system that moves the object around in the project was made by me, using Unreal Blueprints. The system directs objects around the scene using hidden reference points that are activated through a collection. When an actor interacts with an object or the event is called, the collection tells the objects that it holds to activate their sequences.
All assets were provided by the Fab Marketplace.
